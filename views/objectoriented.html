C++ and Java are both caled Object Oriented Programming languages, which means that they incorporate encapsulation, absraction, inheritance, and polymorphism as part of their design principles.

Encapsulation is the act of putting data within an object, that can then only be accessed by that object,or throughthat object. Examples of this are private members for classes, with public getters andsetters for those members.

Abstraction is the act of hiding non-essential informationfrom the user. What this means is that, rather than giving someone access to the a programs inner workings when theyneed to use it, you would only let them know of the functions that perform the actions the program can do.

Inheritance isthe basic concept of classes giving each other access to each others functions or objects. This is done so that the code writer doesn't have to create hundreds of identical objects, which decreases complexity.

Finally, polymorphism is the way that programs tell apart two objects or functions withthe same name, which is necessary in object oriented programming, due to all it's other features.
